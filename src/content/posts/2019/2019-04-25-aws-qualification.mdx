まずは3つの資格に挑戦する予定です。

## AWSの資格
<OG url="https://aws.amazon.com/jp/certification" />

### 種類
次のとおり、AWS の資格は全部で11あります。

<img src="/screenshots/2019/aws-qualification/aws-qualification-list.png" width="100%" alt="AWS資格の種類" />

この中で、一番易しいものとして「クラウドプラクティショナー」という資格があるのですが、それには挑戦せず、その上のアソシエイトレベルの資格に挑戦したいと思っています。

- ソリューションアーキテクト
- ディベロッパー
- SysOps アドミニストレータ

の3つです。

この中だと、ソリューションアーキテクトを最初に取るのがいいらしいと聞いて、ここから挑戦することに。

### 勉強する前の自分の知識
案件で Lambda や EC2 などの一部サービスを使用した経験があった程度。
一から AWS でインフラ構築とかの経験はありませんでした。

## 対策勉強

自分が今のところ行った勉強としては、主に以下の３つです。

### Udemy - 手を動かしながら2週間で学ぶ AWS基本から応用まで
**※2019/04/30で新規購入停止になりました**

AWS の初学者向けの講座で、特別資格に対しての講座ではありませんでしたが、よく使われるサービスを学ぶことができました。
AWS に少し触れた経験がある程度で、勉強するにしてもどこから？という状態だったので、受講してよかったです。
勉強はじめの頃って、知らない用語やサービス名が頻出して混乱しがち...。講座に沿って進めれば混乱も少ないので助かりました。
