先日、Google Analytics に続いて、もう1つの解析ツールである、Google Search Console を導入してみました。
少し記憶が曖昧なのですが、手順をメモしておきます。

## Google Search Console とは？
<OG url="https://search.google.com/search-console/about?hl=ja" />

Google 検索での掲載順位を見たり、どんなクエリでユーザがサイトにアクセスしているかなどを確認できる解析ツールです。
また、サイトマップを認識させることで Google のインデックスに登録し、サイト内のクローラー巡回を促進させるなど、SEO 対策を効率よく進めることができます。

## 導入手順
1. Google Search Console にアクセスして、「今すぐ開始」を選択

<ImageWrapper class="w-[80%]" src="screenshots/2019/google-search-console/search-console-top.png" alt="Google Search Console トップ画面" />

2. Google アカウントにログイン

3. プロパティタイプの選択（今回は「URL プレフィックス」を選択して、サイト URL を入力して進めます）
<ImageWrapper class="w-[80%]" src="screenshots/2019/google-search-console/select-type.png" alt="プロパティタイプの選択画面" />

ちなみにドメインを選択した場合は、「www」のありなしや「http://」「https://」の違いに関わらず、一括での登録・集計が可能となります。
複数の URL でそれぞれ集計したい時はこちらがいいようです。

4. 所有権の確認
   もしログインした Google アカウントで Google Analytics を導入していると、そのアカウントから所有権を確認してくれます。
<ImageWrapper class="w-[80%]" src="screenshots/2019/google-search-console/check-domain.png" alt="所有権をGoogle Analyticsから確認した画面" />

導入していなかった場合は、既定の HTML ファイルをアップロードする、HTML の meta タグをサイトに追加するなど、他の確認方法が選択できます。

5. 所有権の確認方法の追加
ここまでで Google Search Console が使用できるようになりますが、所有権の確認方法は複数あった方がいいようなので、追加します。

メニューの「設定」へ行き、「所有権の確認」を選択。
その他の確認方法の中から、いずれかを選択して表示される手順を行えば追加できます。

### サイトマップの読み込み
サイトマップを登録して Google にサイトを認識させることで、検索結果に表示させやすくできます。

<ImageWrapper class="w-[80%]" src="screenshots/2019/google-search-console/search-console-sitemap.png" alt="Google Search Consoleにサイトマップを登録" />

Jekyll の場合は、`jekyll-sitemap`というプラグインを導入することで、自動で`sitemap.xml`を作成してくれます。
<OG url="https://github.com/jekyll/jekyll-sitemap" />

サイトだと sitemap xml generator というのが有名みたいです。
<OG url="https://www.xml-sitemaps.com" />

### Google Analytics との連携
Google Analytics と連携することで、Google Analytics 側から Google Search Console の機能を使用できます。

所有権の確認の際に、Google Analytics と連携してくれましたが、機能を連携させるところは手動で設定する必要があるようです。

1. Google Analytics 側で「管理」へ
2. プロパティの中の「プロパティ設定」を選択
3. 「Search Console を調整」を選択してビューにサイトの URL を登録する

登録後にプロパティの中のサービス間のリンク設定の中の「すべての商品」を選択して、リンク済みのサービスの中に Search Console があればOK。
<ImageWrapper class="w-[80%]" src="screenshots/2019/google-search-console/analytics-link.png" alt="Google Analyticsと他サービスとのリンク状況" />

これで Google Analytics 側から、Google Search Console の機能が使えるようになりました。
<ImageWrapper class="w-[80%]" src="screenshots/2019/google-search-console/analytics-search-console.png" alt="Google Analytics側からGoogle Search Consoleの機能を使用" />

### プライバシーポリシー
Google Analytics と同様に、情報を収集する解析ツールであるため、プライバシーポリシーの記述が必要になります。
自分の場合は Google Analytics 導入の際に記述した箇所にて付け加える形で書きました。
